# docker model config edit

<!---MARKER_GEN_START-->
Open the config file in the default editor.

The editor is determined by the VISUAL or EDITOR environment variables,
falling back to vi on Unix and notepad on Windows.

### Options

| Name           | Type     | Default | Description                             |
|:---------------|:---------|:--------|:----------------------------------------|
| `-f`, `--file` | `string` |         | use a specific config file              |
| `--global`     | `bool`   |         | use the global (user-level) config file |
| `--system`     | `bool`   |         | use the system-wide config file         |
